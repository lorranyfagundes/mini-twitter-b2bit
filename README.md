# 🐦 Mini Twitter - Teste Técnico

Este projeto é um clone funcional de um "Mini Twitter", desenvolvido como parte de um desafio técnico para a vaga de Estágio - Desenvolvedor Frontend.

## 💻 Tecnologias Utilizadas

* **React** com **TypeScript**
* **Vite** (Build tool)
* **Tailwind CSS** (Estilização e Responsividade)
* **React Query (TanStack Query)** (Gerenciamento de estado de API e Cache)
* **React Hook Form** + **Zod** (Validação de formulários)
* **Lucide React** (Ícones)

## ✨ Diferenciais Implementados

* **Scroll Infinito:** Carregamento automático de posts utilizando `useInfiniteQuery` e `Intersection Observer API`.
* **Busca com Debounce:** Sistema de filtro que aguarda o usuário parar de digitar para realizar a requisição, otimizando o consumo da API.
* **UX Dinâmica:** Interface adaptada para usuários visitantes (sem login) e usuários autenticados.
* **CRUD Completo:** Criação, leitura, edição, exclusão e curtidas de posts.

## 🛠️ Como rodar o projeto

1. Clone este repositório.
2. Certifique-se de que o Backend fornecido esteja rodando na porta `3000`.
3. Instale as dependências:
      ```bash
   npm install
4. Inicie o projeto:
   ```bash
   npm run dev
